Cloudzilla is a minimal, self-hosted Git forge. Repository hosting, pull requests, code review, and an issue tracker — the parts a small team uses, nothing they don't.

Written in Go with a PostgreSQL backend. Ships as a single binary with no external runtime dependencies and embedded migrations. Deploy to any VPS in minutes — no Kubernetes required.
