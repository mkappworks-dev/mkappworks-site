Dev Bench is a local-first desktop app that consolidates the tools developers reach for constantly. The DB explorer connects to Postgres, MySQL, and SQLite — browse schemas, run queries, inspect results, no cloud required.

Also includes: HTTP client, JSON formatter and differ, base64 encoding, hash generation, regex tester, timestamp converter. No accounts, no telemetry, no internet connection required.

Part of the Benchlabs suite of open source developer tools.
