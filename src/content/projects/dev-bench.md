Dev Bench is a local-first desktop utility belt. The DB explorer connects to Postgres, MySQL, and SQLite — browse schemas, run queries, inspect results.

HTTP client, JSON formatter and differ, base64 encoding, hash generation, regex tester, timestamp converter. No accounts, no telemetry, no internet required.

Part of the Benchlabs suite of open source developer tools.
